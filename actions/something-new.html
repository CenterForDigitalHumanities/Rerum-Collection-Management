Begin by naming your New Thing. This label is only for showing in the display and should be descriptive for humans.

Type describes what sort of Thing this is. It is possible for something to be more than one Type, a general one and a more specific Classification. Select the one that best describes this thing.

Location describes where this is or was. It is possible for something to have multiple locations, especially over time. This general relationship is for the one that best describes this thing because it was made here, is stored here, or is about here.
